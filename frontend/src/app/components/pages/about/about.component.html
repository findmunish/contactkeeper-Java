About Contact Keeper

Keep Your Contacts Organized

Contact Keeper is a modern, full-stack application built with Angular 17 and Spring Boot to help you manage your personal and professional contacts efficiently.

Features
  • Add new contacts
  • Edit existing contacts
  • Delete contacts
  • Search and filter
  • Secure authentication
  • Responsive design
Technology Stack
  • Frontend: Angular 17
  • Backend: Spring Boot 3.2
  • Database: MongoDB
  • Authentication: JWT
  • UI Framework: Bootstrap 5
  • Icons: Font Awesome

Built with modern web technologies for the best user experience
